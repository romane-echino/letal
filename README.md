# Letal - Gamehub - Launch games deadly fast


## 🛠️ Technologies

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **UI Components**: HeadlessUI + Lucide
- **Desktop**: Electron
- **Logging**: Sentry

## 🛣️ Roadmap
